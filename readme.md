## Ejercicio 4
- String es un tipo por referencia porque internamente deriva de System.String, que es una clase.
- Algunas de las secuencias de escape que tiene string son:
    * \n: nueva línea
    * \t: tabulación
    * \0: caracter nulo
    * \": comillas dobles
    * \': comillas simples
- Para @: Ignora secuencias de escape y permite escribir rutas a archivos o texto multilínea fácilmente.
- Para $: Permite insertar variables o expresiones dentro del texto usando "{}".
